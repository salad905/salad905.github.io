My Projects

Fake News Detector

    This was a project I worked on for my Data Mining class. All details can be found in our paper.
    Summary:

    1. Abstract

    Misinformation is an ever-lasting relevant issue. To combat this, we aimed to classify news articles as real or fake using natural language processing and machine learning. It focuses on detecting fabricated content based on textual patterns, aiming to support digital literacy and trust in online information.

    2. Methodologies

    Found a Kaggle dataset with about 40k articles. Preprocessing steps included tokenization, lemmatization, and feature extraction using Bag of Words and TF-IDF. Several classifiers were tested, including Logistic Regression, Naive Bayes, and Random Forest, with model performance evaluated through metrics like accuracy, ROC-AUC, and confusion matrix. A Gradio interface was created for interactive predictions based on user-inputted text.

    3. Reflection

    The lines between "fake" and "real" are ambiguous. As much as this project helped familiarize me with NLP tasks, exploratory data analysis, building a front-end interface, and so on, I found that this project is not the most practical.

Deepfake Generation & Detection

    This is my 2025 Summer Project. Summary:

    1. Abstract

    This project addresses the growing threat of deepfake videos by building a machine learning pipeline to detect manipulated facial content. It targets the classification of real vs. fake images, contributing to media authentication efforts.

    2. Methodologies

    Removed background using BiRefNet. Face detection and cropping were performed using MTCNN and facenet-pytorch. Embeddings from a pre-trained FaceNet model were extracted and used to train a classifier. Evaluation included accuracy, confusion matrices, ROC-AUC, and visual inspection of misclassified samples. Currently has a running Gradio interface, planning to expand to a Chrome extension.

    3. Reflection

    Deepfake images are reaching an eery realness. AI will eventually get to the point of complete indistinguishability from real images to the average human eye. I am aware that I can't shoot for anything groundbreaking with this project, but I at the very least hoped for promotion of media literacy and a sense of caution when approaching faces on the Internet.
